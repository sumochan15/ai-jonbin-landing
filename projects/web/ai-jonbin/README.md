# AI Jonbin - 公式ウェブサイト

AI Jonbinの公式ウェブサイトプロジェクト

## 📋 プロジェクト情報

- **プロジェクト名**: AI Jonbin Website
- **種類**: 静的HTML/CSS/JSサイト
- **状態**: 開発中 (In Development)
- **デプロイ先**: Vercel

## 🚀 デプロイ

### Vercel
このプロジェクトはVercelにデプロイされます。

**セットアップ手順**:
1. GitHubにプッシュ
2. Vercelダッシュボードで新しいプロジェクトをインポート
3. プロジェクトのRoot Directoryを `projects/web/ai-jonbin` に設定
4. デプロイ

## 🛠️ 技術スタック

- Vanilla JavaScript

## 🔧 ローカル開発

静的サイトなので、`index.html`をブラウザで直接開くか、ローカルサーバーを使用してください。

```bash
# Python3のローカルサーバーを使用する場合
cd /Users/yohei/Documents/vault/projects/web/ai-jonbin
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス
